# PyGaze — captured summary

## Description
"An open-source, cross-platform toolbox for minimal-effort programming of eye
tracking experiments." Simplifies building gaze-contingent and non-gaze-
contingent studies by providing one unified interface across eye-tracker
hardware.

## Supported trackers
Advertised as hardware-agnostic across major research eye-trackers (EyeLink,
SMI, Tobii, GazePoint, etc.). The fetched README did not enumerate brands
explicitly, but the unified-API design and per-vendor backends are the point of
the project.

## Features
- Unified API standardizing access across different eye-tracking hardware.
- Real-time / online event detection for gaze-contingent experiments.
- Cross-platform (multiple OSes).
- Ships core `pygaze` module, OpenSesame plugins, examples, and helper libs.
