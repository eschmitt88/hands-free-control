# macOS Voice Control Command Reference

## Core Command Categories

**Navigation & Control**: Users can "say commands" like "Open Mail," "Scroll Down," or "Click Done" to control their Mac hands-free. Commands work best with approximately half-second pauses between successive requests.

**Overlay Systems**

The system offers two primary labeling methods:

1. **Item Labels**: Say "Show names" or "Show numbers" to display identifiers next to screen elements. Users then reference these to interact—for example, commanding "Click 5" to activate labeled item number five.

2. **Grid Overlay**: Say "Show grid" to superimpose a numbered grid across the screen (or "Show window grid" for the active window only). Users drill down by speaking grid numbers for finer precision across multiple levels.

## Text Entry Modes

Voice Control provides three specialized text modes:

- **Dictation Mode** (default): Words are transcribed automatically unless recognized as commands
- **Spelling Mode**: Character-by-character entry using phonetic alphabet codes (e.g., "Alfa Bravo Charlie" for "abc")
- **Command Mode**: Only commands execute; non-command speech is ignored
- **Swift Mode**: Formats Swift programming code automatically

## Additional Capabilities

Users can retrieve "Show commands" to display context-sensitive available actions, customize vocabulary, integrate with VoiceOver accessibility features, and access an interactive tutorial through System Settings.
