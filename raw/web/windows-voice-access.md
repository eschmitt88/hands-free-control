# Windows 11 Voice Access Command Summary

## Overview
Voice Access is an on-device feature enabling PC control and text authoring through voice commands without internet connectivity. Available in Windows 11 version 22H2+ and supports English, Spanish, French, German, Chinese, and Japanese.

## Core Command Categories

**Manage Voice Access**
Commands to activate ("Voice access wake up"), deactivate ("Voice access sleep"), and access settings include toggling between command-only, dictation-only, and default modes.

**App & Window Control**
Users can "Open <app name>," "Close <app name>," "Switch to <app name>," and perform window operations like minimize, maximize, and snap to specific screen directions.

**UI Interaction**
"Click <item name>," "Double-click <item name>," and "Scroll <direction>" enable navigation. Users can also "Show numbers" for overlay-based element selection and "Show grid" for precision targeting across screen areas.

**Text Dictation & Editing**
Direct voice input inserts text. Commands include "Delete that," "Copy that," "Paste," and "Undo that" for content manipulation. Text selection uses "Select that," "Select <text>," or "Select previous <count> words."

**Formatting**
Users apply styles with "Bold <text>," "Italicize <text>," "Underline <text>," and case modifications like "Uppercase <word>" or "Lowercase <word>."

**Navigation & Positioning**
"Go to top," "Move to end of document," "New paragraph," and directional cursor movement commands ("Move left <count> words") enable precise text positioning.

**Punctuation & Symbols**
Voice commands insert marks: "Period," "Comma," "Question mark," plus symbols like "Asterisk," "Dollar sign," and "Copyright sign."
