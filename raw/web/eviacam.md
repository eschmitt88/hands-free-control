# eViacam — captured summary

## What it is
eViacam is a mouse-replacement application that moves the pointer as the user
moves their head. It runs on standard PCs with an ordinary webcam and requires
no additional hardware. Positioned for accessibility (motor disability) and for
mitigating repetitive strain injury.

## How it works
Uses webcam-based head-tracking: it monitors head movement in the camera feed
and translates that motion into pointer movement across the screen. Fully
hands-free / wireless (works with any standard webcam).

## Key features (customizable parameters)
- Pointer speed and motion acceleration adjustment
- Smoothness control for fluid cursor motion
- Dwelling time configuration for click activation (dwell click)
- Hands-free operation, no manual input needed
- Integrated configuration wizard for setup

## Supported platforms
- Windows Vista / 7 / 8 / 10 (primary distribution)
- GNU/Linux (Debian 10+ / Ubuntu 18.10+; other distros via source build)
- Android via the "EVA Facial Mouse" variant
