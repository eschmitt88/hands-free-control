## Captured content

Boggle is a BCI web browser for individuals with severe motor impairment (e.g.
ALS, stroke), enabling internet access using only brain signals (SSVEP).

- **Architecture:** built with Electron (open-source, cross-platform; Node.js +
  Chromium). SSVEP input modality; uses commercial EEG headsets such as the
  **dry-electrode DSI-VR/VEP headset by Wearable Sensing**. Ships a
  customisable, empirically verified SSVEP stimulus generator built with native
  web technologies.
- **Performance (online, asynchronous BCI):** global mean classification
  accuracy **90.98%** and ITR **29.58 BPM (bits per minute)**.
- **Goal:** reduce entry barriers to BCI web browsing (lower complexity + cost);
  architecture aimed at out-of-lab use.
