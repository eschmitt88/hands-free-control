# VoicePilot Paper Summary

## Core Details
**Title:** VoicePilot: Harnessing LLMs as Speech Interfaces for Physically Assistive Robots

**Authors:** Akhil Padmanabha, Jessie Yuan, Janavi Gupta, Zulekha Karachiwalla, Carmel Majidi, Henny Admoni, Zackory Erickson

**arXiv ID:** 2404.04066 (v2 submitted July 17, 2024)

**Venue:** ACM Conference (DOI references UIST 2024: 10.1145/3654777.3676401)

**Disciplines:** Robotics, Computational Linguistics, Human-Computer Interaction

## Abstract Summary
The research addresses how LLM-based speech interfaces can serve individuals with motor impairments needing assistance with daily activities. The abstract notes that "frameworks for integrating LLMs as interfaces to robots...fail to incorporate human-centric considerations which are essential while developing assistive interfaces."

## Methodology
- **Iterative Design:** Three testing stages using a feeding robot system
- **Final Evaluation:** Study with 11 older adults at an independent living facility
- **Data Collection:** Both quantitative metrics and qualitative feedback

## Key Contributions
The team developed a systematic framework that bridges LLM capabilities with assistive robotics while emphasizing user-centered design. They generated design guidelines specifically addressing speech interface requirements for assistive robots.
