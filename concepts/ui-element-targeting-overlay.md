# UI Element Targeting Overlay
## Definition
Overlaying visual markers on screen elements so a user can reference them by voice to click or act on them.
## Why it matters here
Overlays give voice a precise addressing scheme for arbitrary UI, the complement that turns coarse gaze pointing into reliable selection.
