# LLM Planning
## Definition
Sequential task decomposition, reasoning, and memory that let an LLM agent carry out multi-step goals.
## Why it matters here
Real hands-free requests ("clean up my inbox") span many steps; planning is what turns a single high-level voice command into a correct sequence of desktop actions.

- [[llm-agent-tool-use]] — planning sequences the tool calls that actually execute each step
