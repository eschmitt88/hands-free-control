# Spelling Mode
## Definition
Character-by-character voice entry, often via a phonetic alphabet, for proper nouns and code tokens that ASR reliably mishears.
## Why it matters here
No ASR gets unusual identifiers right, so a phonetic spelling fallback is what makes hands-free entry of names, passwords, and code tokens actually workable.
## Connections
- [[dictation-mode]] — spelling is the fine-grained fallback when word-level dictation fails
