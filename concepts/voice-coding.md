# Voice Coding
## Definition
Writing and editing source code by voice through structured, syntax-aware commands rather than raw dictation.
## Why it matters here
Coding is a demanding, high-value hands-free target: its dense symbols and structure stress-test the command and mapping layers harder than prose, making it a sharp proving ground for the system.
## Connections
- [[command-grammar]] — voice coding relies on a specialized, syntax-aware grammar
- [[speech-to-action-mapping]] — code edits are structured actions mapped from spoken intent
