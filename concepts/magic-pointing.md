# MAGIC Pointing
## Definition
Manual And Gaze Input Cascaded: gaze warps the cursor to the gaze region for coarse positioning, then a separate actuator completes fine selection.
## Why it matters here
It is the foundational technique the project generalizes — replacing MAGIC's manual fine stage with a voice command yields a fully hands-free version of the same cascade.

- [[gaze-supported-interaction]] — MAGIC is the canonical example of gaze supporting rather than driving input
