# Number-overlay Targeting
## Definition
Labeling clickable elements with numbers, as in "show numbers", so the user selects one by speaking its number.
## Why it matters here
It is a shipping, robust way to select any UI element by voice, a strong baseline the gaze-plus-voice approach must match or streamline.
