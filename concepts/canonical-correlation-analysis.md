# Canonical Correlation Analysis
## Definition
CCA, a reference-template correlation method that detects which SSVEP stimulus frequency a user is attending; it is the standard on-device SSVEP decoder.
## Why it matters here
It represents the mature, low-compute end of EEG decoding, but sits within the project's parked EEG track and is tracked only as a reference for any future SSVEP experiment.
## Connections
- [[ssvep]] — CCA is the canonical decoder for the SSVEP paradigm
