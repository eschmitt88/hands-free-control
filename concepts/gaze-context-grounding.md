# Gaze Context Grounding
## Definition
Using where the user is looking as the contextual anchor that grounds a voice command or query to a specific object or region.
## Why it matters here
It turns gaze into the shared referential context between user and system, so spoken intent lands on the right target without an explicit manual selection step.
## Connections
- [[deictic-reference]] — gaze grounding supplies the referent for deictic commands
- [[implicit-anchoring]] — grounding via gaze is a specific implicit anchoring signal
