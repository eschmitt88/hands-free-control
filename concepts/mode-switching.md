# Mode Switching
## Definition
Toggling an interaction mode — such as entering multi-select — an action best suited to discrete, infrequent triggers like a voice command.
## Why it matters here
Discrete mode changes are exactly where voice outperforms gaze, so the project should route mode toggles to speech and keep gaze on continuous pointing.
## Connections
- [[multi-object-selection]] — a mode whose entry is a switch of this kind
- [[voice-mode-switching]] — voice is the preferred trigger for discrete mode changes
